# EduSuite AI v0.3 Alpha

## Novedades

- Nuevo centro de comando con indicadores vivos.
- Acciones rápidas para Personas, Cursos y Organización.
- Sidebar profesional con agrupación por áreas.
- Topbar con breadcrumb, búsqueda visual y perfil de usuario.
- Navegación móvil mediante menú lateral.
- Panel de estado de capacidades.
- Roadmap inmediato visible desde Inicio.
- Identidad visual consolidada para la release 0.3.

## Validación

- Ejecutar `npm install`.
- Ejecutar `npm run build`.
- Ejecutar `npm run dev`.
- Revisar Inicio, Personas, Cursos, Organización, Datos y Design System.
