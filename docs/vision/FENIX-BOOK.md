# Fénix Book

## Propósito
Devolver tiempo a quienes educan.

## Principios
1. Una capacidad debe ahorrar más tiempo del que toma aprenderla.
2. Nunca obligaremos a un docente a realizar dos veces la misma tarea.
3. La IA propone; las personas deciden.
4. Los datos pertenecen a cada institución.
5. La seguridad y la recuperación son requisitos.
