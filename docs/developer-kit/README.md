# EduSuite AI Developer Kit v0.1

Este kit define cómo se construyen nuevas capacidades dentro de Proyecto Fénix.

## Incluye
- Convenciones de nombres
- Flujo de desarrollo
- Plantilla de módulos
- Prompt base para asistentes de IA
- Generador automático de módulos

## Uso rápido

```bash
node scripts/create-module.mjs nombre-del-modulo
```
