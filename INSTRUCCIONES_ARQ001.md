# ARQ001 — Layout Profesional

1. Copia todo el contenido del paquete dentro de la raíz de `proyecto-fenix`.
2. Acepta reemplazar `package.json` y `src/app/App.tsx`.
3. Abre `src/main.tsx` y debajo de `import './styles/edupeople.css';` agrega `import './styles/layout.css';`.
4. Abre terminal CMD y ejecuta `npm install`.
5. Ejecuta `npm run dev`.
6. Prueba `/inicio`, `/personas`, `/cursos` y `/organizacion`.
7. Commit: `ARQ001 - Layout profesional y navegación`.
