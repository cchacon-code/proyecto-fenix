# Instalación de EduSuite AI 0.5

1. Conserva una copia de respaldo del repositorio actual.
2. Copia el contenido de esta release dentro del repositorio original.
3. Conserva la carpeta oculta `.git` del repositorio original.
4. En una terminal CMD ejecuta:

```cmd
npm install
npm run dev
```

5. Ingresa y abre `Foundation 0.5` desde el menú Sistema.
6. Comprueba `Ctrl + Shift + M`: debe indicar cero errores.
